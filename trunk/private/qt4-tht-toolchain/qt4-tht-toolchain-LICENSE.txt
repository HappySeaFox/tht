This toolchain contains the following libraries and binaries:

* Qt4
* MinGW (under mingw/)
* 7-zip (under bin/)
* UPX (under bin/)
* JOM (under bin/)
* qt-path-corrector (under extra/)
* openssl (under extra/openssl/)

You need to accept the appropriate licenses for that projects
to use this toolchain. You can find the files with licenses in the installed
directory. If you don't accept that licenses, please remove the toolchain.
